# 对比 Enzyme 和 react-testing-library

## 概念

### Enzyme

Enzyme 是 Airbnb 推出的流行的测试库，Enzyme 的 API 旨在通过模仿 jQuery 的 API 来实现直观和灵活的 DOM 操作和遍历。
